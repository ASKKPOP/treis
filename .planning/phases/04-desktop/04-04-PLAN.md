**1. Generate macOS app icon (.icns) from placeholder PNG:**

If `apps/desktop/build/icon.png` is a valid 1024x1024 PNG, convert to `.icns`:
```bash
cd apps/desktop/build
mkdir -p icon.iconset
sips -z 16 16 icon.png --out icon.iconset/icon_16x16.png
sips -z 32 32 icon.png --out icon.iconset/icon_16x16@2x.png
sips -z 32 32 icon.png --out icon.iconset/icon_32x32.png
sips -z 64 64 icon.png --out icon.iconset/icon_32x32@2x.png
sips -z 128 128 icon.png --out icon.iconset/icon_128x128.png
sips -z 256 256 icon.png --out icon.iconset/icon_128x128@2x.png
sips -z 256 256 icon.png --out icon.iconset/icon_256x256.png
sips -z 512 512 icon.png --out icon.iconset/icon_256x256@2x.png
sips -z 512 512 icon.png --out icon.iconset/icon_512x512.png
sips -z 1024 1024 icon.png --out icon.iconset/icon_512x512@2x.png
iconutil -c icns icon.iconset -o icon.icns
rm -rf icon.iconset
```

If `icon.png` is a placeholder (not 1024x1024), create a minimal valid 1024x1024 PNG first using ImageMagick or a simple script. The icon is a placeholder for Phase 4.

**2. Verify `electron-builder.config.json`** is correct per RESEARCH.md:
```json
{
  "appId": "io.treis.desktop",
  "productName": "Treis",
  "directories": { "buildResources": "build", "output": "release" },
  "files": ["out/**/*", "node_modules/**/*", "!node_modules/**/test/**", "!node_modules/**/*.map"],
  "mac": {
    "category": "public.app-category.developer-tools",
    "hardenedRuntime": false,
    "target": [{ "target": "dmg", "arch": ["universal"] }],
    "icon": "build/icon.icns"
  },
  "dmg": {
    "title": "Treis ${version}",
    "artifactName": "Treis-${version}-macOS.dmg"
  }
}
```

**3. Ensure `package.json` has correct `build:mac` script:**
```json
"build:mac": "pnpm build && electron-builder --config electron-builder.config.json --mac --universal"
```

Note: If universal build fails (common without code signing), fall back to current arch only:
```bash
pnpm --filter @treis/desktop exec electron-builder --config electron-builder.config.json --mac
```

**4. Build the full monorepo first, then package:**
```bash
cd /Users/desirey/Programming/src/treis
pnpm build                              # Build all packages
pnpm --filter @treis/desktop build:mac  # Package DMG
```

**5. Verify DMG exists:**
```bash
ls -la apps/desktop/release/*.dmg
```

The DMG should be at `apps/desktop/release/Treis-0.0.1-macOS.dmg` or similar.
